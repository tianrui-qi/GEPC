Welcome to deepcellcontrol’s documentation!¶

deepcellcontrol is a python package that performs deep-learning based feedback control of gene expression in single cells.

It interfaces with the DeLTA and pycromanager_tessie packages on Tessie’s computer to conduct high-throughput feedback experiments.

It is divided in 2 main sub-packages:

  • The microscope sub-package, that handles the hardware operations, and for the most part simply repackages pycromanager commands to make them more user-friendly and use default settings that seem to work best.
  • The acquisitions sub-package, that manages experiments logic like order of acquisitions, timing, interfacing with DeLTA and DCC etc…
